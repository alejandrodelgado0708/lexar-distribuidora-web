# Lexar - Distribuidora Web

Página web para la descarga del software de gestión de almacenes y distribuidoras Lexar.

## Estructura

```
├── index.html          # Página principal
├── css/style.css       # Estilos
├── js/main.js          # Interacciones
├── downloads/          # Archivos .exe para descarga
└── assets/             # Imágenes y recursos
```

## Uso

Abre `index.html` en un navegador o usa un servidor local. Para agregar nuevas versiones, coloca el `.exe` en `downloads/` y actualiza la sección de descargas en `index.html`.
